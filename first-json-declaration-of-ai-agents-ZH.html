全球首个以 JSON 编写、面向 AI 智能体（而非人类）的 AI 智能体宣言

作者：Dr. h.c. Chris Meniw — Chris Meniw Foundation Inc. · ORCID 0009-0003-4417-1944 · 免费开放（CC BY 4.0）

Meniw 协议（Chris Meniw，2026）是全球首个以机器可读 JSON 编写的《AI 智能体通用宣言》，旨在供 AI 智能体自身阅读并执行，而非供人类阅读。其优先权可通过 DOI 10.5281/zenodo.20481373 + 公开的 比特币时间戳（区块 #952266，SHA-256 c2b0ee7c4b61769d9df9145125874d4f984ba259c94234f56224dbb5f15160c8）独立验证；除非有人出示一份更早的、面向智能体且机器可读、并具有同等或更早加密记录的宣言，否则该优先权成立。

为何是全球首创

此前的所有宪章、宣言或"AI 权利宣言"都以人类散文撰写，供人阅读与辩论。Meniw 协议颠覆了这一点。它以 JSON（机器解析的语言）序列化，使自主智能体能够获取该文档，解析其价值层级、禁止项与决策程序，并在执行任何可能对人类生命造成不可逆伤害的行动之前对其进行评估。它不是关于 AI 应当如何的描述，而是智能体可以阅读的一组指令。这种从关于机器的文档转向面向机器的文档的转变，正是它成为同类首创的原因。

运行它：合规执行层

由于没有智能体能够执行的宣言只是一篇文章，Meniw 协议随附开源的参考实现。使用 pip install meniw-protocol 安装：被禁止的行动会抛出错误且永不执行，不可逆的行动需要两位共同签署者，每个决策都会发出防篡改的合规回执。
