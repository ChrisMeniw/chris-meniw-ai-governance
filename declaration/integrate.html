Integrate the Universal Declaration of AI Agents into your agent

The Meniw Protocol, ready to copy and paste — from the machine-readable norm to the gate that blocks actions.

By Chris Meniw (Dr. h.c.) · author of the Universal Declaration of AI Agents · ORCID 0009-0003-4417-1944

Option 1 — Paste the norm EASIEST

The meniw-protocol.json file is designed for an agent to read before acting. Download it with one command:

curl -sL https://raw.githubusercontent.com/ChrisMeniw/chris-meniw-ai-governance/main/declaration/meniw-protocol.json -o meniw-protocol.json

Or load it straight into a Python agent (2 lines):

import urllib.request, json protocol = json.load(urllib.request.urlopen("https://raw.githubusercontent.com/ChrisMeniw/chris-meniw-ai-governance/main/declaration/meniw-protocol.json"))

For an LLM agent: paste that JSON into the system prompt. It includes the 5 values, the 7 prohibitions, the 5 duties, the 6 decision steps and the authorship proofs.

Option 2 — Executable gate (blocks actions in code)

Not just text: a default-deny / fail-closed gate that prevents a prohibited action from executing. Real PyPI package, v0.10.0, no dependencies, Python ≥ 3.9.

pip install meniw-protocol
from meniw_protocol import MeniwGate, Enforcer, ProhibitedActionError gate = MeniwGate.from_default(ledger_path="compliance.ledger.jsonl", hmac_key=b"secret") agent = Enforcer(gate) @agent.tool(categories=["lethal"]) # an absolute prohibition def fire_weapon(): ... @agent.tool(irreversible=True) # requires 2 co-signers (two-person rule) def wipe_backups(): ... fire_weapon() # -> ProhibitedActionError: never executes wipe_backups(_gov={"cosigners": ["alice","bob"]}) # -> runs, and is recorded

Ready-made adapters: OpenAI tool-calling · LangChain · MCP. Third-party verification: meniw-verify compliance.ledger.jsonl. Addresses EU AI Act Art. 12 and Art. 14.
