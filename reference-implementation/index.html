Meniw Protocol — Reference Implementation

Open, runnable reference implementation of the Meniw Protocol (Universal Declaration of AI Agents) by Chris Meniw: a runtime governance layer with a default-deny gate, dual-signature for irreversible actions and verifiable compliance receipts. Installable: pip install meniw-protocol.
